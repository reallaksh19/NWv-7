  rolling history
  dedup
  event clustering
  anchors
  angles
  ranking
  digest integrity
  static output

Scope
-----
Allowed:
  - add all-in-one Insight benchmark runner;
  - add contract schema tests;
  - add static output tests;
  - add PR workflow;

Not allowed:

Benchmark runner
----------------
benchmarks/insight/runInsightBenchmarks.mjs runs:
  Wave 4 Dedup
  Wave 5 Event Clustering
  Wave 6 Anchors
  Wave 7 Angles
  Wave 8 Ranking

test_insight_all.mjs runs:
  contract schemas
  rolling history integrity (optional if data not yet generated)
  insight benchmarks
  digest integrity (optional if data not yet generated)

test_insight_ci_static.mjs requires actual generated files:
  insight_digest.json
  insight_diagnostics.json
  top_story_anchors.json
  insight_latest.json
  insight_history.json
  insight_source_health.json

PR workflow
-----------
.github/workflows/insight_ci.yml should run on PRs touching:
  src/pages/InsightPage.jsx
  src/components/insight/**
  src/adapters/insightDigestFetcher.js
  src/insight/**
  scripts/insight_worker/**
  scripts/test_insight_*.mjs
  benchmarks/insight/**
  public/newsdata/schema/**
  package.json
  .github/workflows/insight_ci.yml
  .github/workflows/insight_refresh.yml

CI steps:
  checkout
  setup Node
  setup Python
  npm ci
  pip install -r requirements.txt
  npm run test:insight-contracts
  npm run benchmark:insight
  npm run build

Refresh workflow gates
----------------------
After building snapshot and digest:
  npm run test:insight-contracts
  npm run test:insight-history
  npm run test:insight-digest
  npm run test:insight-static
  npm run benchmark:insight

Only commit JSON if all tests pass.
